== Description ==
This plugin is used to connect a woocommerce store to the traction apps backend. To use this plugin, the user requires an account with Traction Apps.


== Installation ==
1. Upload `traction apps folder` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings >> Traction Connection Setting
4. Enter Username and Password to authenticate user account
5. Select store
6. Authorize App
